4.3绑定内联样式

大多数情况下，直接写一长串的样式不便于阅读和维护，所以一般写在data或computed里，以data为例： 

                <div id="app">
                    <div :style="styles">文本</div>
                </div>
                <script>
                    var app = new Vue({
                        el: '#app',
                        data: {
                            styles: {
                                color: 'red',
                                fontSize: 14 + 'px'
                            }
                        }
                    })
                </script>
应用多个样式对象时，可以使用数组语法： <div :style="[styleA, styleB]">文本 </div> 在实际业务中，:style的数组语法并不通用，因为往往可以写在一个对象里面；而较为常用的应当是计算属性。
另外，使用:style时，Vue.js会自动给特殊的css属性名称增加前缀，比如transform
